### 🚀 Portranger CMS

**Portranger CMS** is a lightweight, open-source engine designed to help you build stunning portfolios and professional public landing pages with ease. Built by **Athfal Ahmad**, it eliminates the overwhelming complexity of heavy platforms like WordPress, delivering exactly what you need right out of the box.

It is highly optimized for non-technical users. It features an automated, web-based installation setup wizard that configures your database and admin credentials instantly without touching any code.

* * *

### ✨ Key Features

*   **Zero WordPress Complexity:** Light, fast, and purpose-built for portfolios and landing pages.
*   **Built-in Admin Panel:** Manage all your front-end content, copy, and images from a clean dashboard.
*   **Web-Based Installer CLI:** Just spin up the server, open your browser, and fill in the form.
*   **Ready Out of the Box:** No bloated plugins, theme configurations, or heavy databases required.
*   SEO features and meta keywords to keep you top in the line.

* * *

### 💻 System Requirements

Before running Portranger CMS, make sure your computer has the following tools installed:

*   **PHP** (Lower version would work but version 8.0 or higher recommended)
*   **MySQL Server** (Running locally, e.g., via XAMPP, MAMP, or native MySQL installation)

### 🚀 Quick Start Installation

Setting up your website takes less than two minutes. Follow these simple steps:

### 1\. Download the Engine

You can download Portranger CMS in one of two ways:

*   **Option A (Via Git):** Clone this repository using your terminal:
    
    bash
    
        git clone https://github.com
        cd portranger-cms
        
    
    Use code with caution.
    
*   **Option B (Manual ZIP):** Go to the **Releases** section on this GitHub page, download the latest source code ZIP file, and extract it on your computer.

### 2\. Start the PHP Local Server

Open your terminal or command prompt inside the project folder and run the native PHP server command:

bash

    php -S localhost:8000
    

Use code with caution.

### 3\. Complete the Setup Wizard

1.  Open your web browser and navigate to: **`http://localhost:8000`**
2.  The **Portranger Web Installer** will launch automatically.
3.  Fill in your MySQL database credentials (host, username, database name, and password).
4.  Create your Admin Dashboard account credentials.
5.  Click **Install**, and your website is ready to use!
